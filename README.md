# Nithish Sagar - Full Stack Developer Portfolio

A modern, animated portfolio website built with React, TypeScript, Tailwind CSS, and Vite.

## Features

- Responsive design with Tailwind CSS
- Smooth animations and transitions
- Hero section with jello letter animations
- Skills categorized by tech stack
- Project showcase
- Resume download functionality
- Contact information
- Modern gradient effects

## Tech Stack

- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS v3
- **Build Tool**: Vite 7.2.4
- **Animations**: Custom CSS keyframes

## Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Opens at `http://localhost:5173/`

### Build

```bash
npm run build
```

Generates optimized production build in `dist/` folder
